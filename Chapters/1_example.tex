		An itemized list:

		\begin{itemize}
			\item An item.
			\item Another item.
			\item Final item.
		\end{itemize}

		An enumerated list:

		\begin{enumerate}
			\item First item.
			\item Second item.
			\item Third item.
		\end{enumerate}
